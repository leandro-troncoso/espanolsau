SAUERKRAUT fue un proyecto de aplicación desarrollado para empleados de un restaurante alemán. Esta tarea incluyó branding, diseño UX/UI y diseño de aplicaciones.



La aplicación permite guardar las recetas del restaurante para consultarlas en cualquier momento, abarcando tanto el proceso de puesta en marcha como el proceso de expedición y emplatado. Además, cuenta con un sistema de alarma para los distintos procesos de las recetas.



Esta aplicación permitiría una gestión ordenada de un servicio gastronómico altamente exigente y una adecuada gestión del tiempo, evitando retrasos o problemas en la preparación de los platos.
